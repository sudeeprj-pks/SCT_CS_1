## ğŸ” What is Caesar Cipher?

The Caesar Cipher is a **substitution cipher** where:

* Each letter in the plaintext is shifted forward by a fixed number (key/shift)
* Decryption shifts the letters backward by the same number

Example with shift = 3:

```
Plaintext : HELLO
Ciphertext: KHOOR
```
