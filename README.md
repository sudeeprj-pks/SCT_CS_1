#  Caesar Cipher Text Encryption Tool (Python)

A simple Python program that encrypts and decrypts text using the **Caesar Cipher algorithm**.  
It allows the user to input a message and a shift value to perform the encryption and decryption process.

---

##  What is Caesar Cipher?

The Caesar Cipher is one of the earliest and simplest methods of encryption.  
It works by shifting each letter of the plaintext by a fixed number of positions down the alphabet.

For example, with a shift of **3**:
Plaintext: HELLO
Ciphertext: KHOOR
